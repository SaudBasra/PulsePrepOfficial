Import questions to the general question bank

Question Bank CSV Requirements:

  • Required: question_text, option_a, option_b, option_c, option_d
  • Classification: degree, year, block, module, subject, topic
  • Answer: correct_answer (A, B, C, D, or E)
  • Optional: option_e, difficulty, explanation
  • Images: question_image (with question), image (with explanation)
  • Image Format: Filename only (e.g., heart_anatomy.jpg)
  • Note: question_image displays with MCQ options, image displays with explanation

Model Paper CSV Requirements:

  • Required: question_text, option_a, option_b, option_c, option_d
  • Paper: paper_name (e.g., "Final Exam 2024")
  • Classification: degree, year, module, subject, topic
  • Answer: correct_answer (A, B, C, D, or E)
  • Optional: option_e, difficulty, explanation, marks
  • Images: paper_image (with question), image (with explanation)
  • Image Format: Filename only (e.g., ecg_reading.png)
  • Note: paper_image displays with MCQ options, image displays with explanation
